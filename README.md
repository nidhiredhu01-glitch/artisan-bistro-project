# 🍽️ The Artisan Bistro – Digital Menu System

## 📌 Project Overview

The **Artisan Bistro Digital Menu System** is a modern web-based menu designed to replicate the elegance of a fine-dining experience in digital form.
It combines **HTML5 and CSS3** to create a visually appealing, responsive, and user-friendly interface.

---

## 🎯 Objectives

* Create a stylish digital menu interface
* Ensure responsiveness across devices
* Maintain semantic and clean code
* Provide a smooth user experience

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Playfair Display)
* Flexbox Layout

---

## ✨ Features

* 🌙 Dark mode aesthetic design
* 🍴 Chef’s Signature Section with auto-numbering
* 📱 Responsive layout (mobile + desktop)
* 🎨 Elegant typography and color scheme
* 📋 Organized menu categories
